# goni-mobile
for mobile push notification and simple chart

### We need react-native-cli
Install with the following command:
```bash
npm install -g react-native-cli
npm install -g rnpm
```

### iOS build
Build ios with the following command:
```bash
npm install
rnpm link
react-native run-ios
```
